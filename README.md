# Web 订单数据录入系统

基于 Flask 的轻量级订单管理系统，适用于包装材料等批发业务的日常开单。

## 功能

- **货物清单管理** — 支持多套价格清单（标准/VIP/专属），一键复制清单快速创建差异化定价
- **客户管理** — 每个客户关联一个货物清单，下单时自动匹配对应价格
- **订单录入** — 选客户即联动货物和价格，添加后自动按客户筛选，连续录入无需重复选择
- **订单查询** — 按客户（支持正则）、货物、日期范围筛选，合计金额置顶显示
- **数据导入导出** — 支持 CSV / Excel 导出，Excel 多 Sheet 全量导出，CSV/Excel 文件导入

## 数据模型

```
货物清单 (price_lists)          客户 (customers)
  id, name                       id, name, list_id
  ┌──────────────┐               ┌────────────────────┐
  │ 1  标准价格   │◄──────────── │ 华盛陶瓷    list=1  │
  │ 2  VIP价格    │◄──┐         │ 顺达电子    list=2  │
  │ 3  宏峰专属   │◄┐ └──────── │ 鑫源塑胶    list=2  │
  └──────┬───────┘ └────────── │ 宏峰一厂    list=3  │
         │                      └────────────────────┘
         ▼
货物明细 (products)
  id, list_id, name, unit, price
  同一清单内支持同名货物按"套"和"个"分别定价
```

## 快速启动

```bash
# 安装依赖
uv sync

# 启动（二选一）
uv run web-order
uv run python order_management/app.py

# 访问 http://127.0.0.1:5001
```

## 技术栈

- 后端：Python / Flask
- 前端：原生 HTML + JavaScript（单页面，无框架）
- 存储：CSV 文件（轻量，无需数据库）
- Excel：openpyxl
